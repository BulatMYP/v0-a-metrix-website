## ✅ Фаза 5: Проверка качества

### Техническое SEO

- [ ] **Проверить Mobile Friendly**
  - [ ] Открыть в Chrome DevTools
  - [ ] Переключиться на мобильный вид
  - [ ] Все страницы хорошо выглядят

- [ ] **Проверить Page Speed**
  - [ ] https://pagespeed.web.dev
  - [ ] Ввести: https://flgmn.tech
  - [ ] Mobile Score > 50
  - [ ] Desktop Score > 70

- [ ] **Проверить Core Web Vitals**
  - [ ] LCP < 2.5s (Largest Contentful Paint)
  - [ ] FID < 100ms (First Input Delay)
  - [ ] CLS < 0.1 (Cumulative Layout Shift)

- [ ] **Проверить Structured Data**
  - [ ] https://schema.org/validator
  - [ ] Ввести URL: https://flgmn.tech
  - [ ] Нет ошибок структурированной разметки

### Контент SEO

- [ ] **Проверить Title tags**
  - [ ] Каждая страница имеет уникальный title
  - [ ] Title содержит основное ключевое слово
  - [ ] Длина 50-60 символов

- [ ] **Проверить Meta descriptions**
  - [ ] Каждая страница имеет уникальное description
  - [ ] Description содержит ключевое слово
  - [ ] Длина 150-160 символов

- [ ] **Проверить H1 теги**
  - [ ] На каждой странице есть один H1
  - [ ] H1 соответствует title
  - [ ] H1 не дублируется на других страницах

- [ ] **Проверить Alt text для изображений**
  - [ ] Все изображения имеют alt text
  - [ ] Alt text описательный
  - [ ] Содержат ключевые слова (где логично)
