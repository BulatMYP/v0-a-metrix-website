## 3. Schema.org Разметка

Реализована расширенная структурированная разметка:

### Organization Schema
- Название компании, альтернативные названия
- URL, логотип, описание
- Контактная информация
- Адреса офисов (Москва, Казань)
- Социальные сети (LinkedIn, Instagram, Telegram)

### LocalBusiness Schema
- Информация о местоположении и услугах
- Телефон, email, часы работы
- Области обслуживания

### EducationalOrganization Schema
- Информация об образовательных программах
- Предметы обучения: запуск стартапа, финансовое моделирование, AI интеграция

## 4. Файлы Robots.txt и Sitemap.xml

### robots.txt
- Оптимизирован для всех основных поисковых роботов
- Специальные инструкции для Google, Yandex, Bing, Mail.ru
- Указаны sitemaps и параметры crawl-delay

### sitemap.xml
- Включены все основные страницы сайта
- Установлены приоритеты:
  - Главная: 1.0 (самый высокий приоритет)
  - Страницы сервиса: 0.8
  - Прочие страницы: 0.5

## 6. Безопасность и производительность

### Security Headers (vercel.json)
- X-Content-Type-Options: nosniff
- X-Frame-Options: SAMEORIGIN
- X-XSS-Protection: 1; mode=block
- Referrer-Policy: strict-origin-when-cross-origin

### Кеширование
- sitemap.xml: 3600 сек с stale-while-revalidate
- robots.txt: 3600 сек с stale-while-revalidate

## 8. Технические параметры

### Core Web Vitals
- Убедитесь, что время загрузки страницы < 2.5 сек
- LCP (Largest Contentful Paint): < 2.5s
- FID (First Input Delay): < 100ms
- CLS (Cumulative Layout Shift): < 0.1

### Мобильная оптимизация
- Сайт полностью адаптирован для мобильных устройств
- Используется viewport meta тег
- Оптимизированы шрифты и CSS для мобильных
