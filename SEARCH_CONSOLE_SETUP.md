# Инструкция по настройке в поисковых системах

## Google Search Console

### Шаг 1: Добавление сайта
1. Перейдите на https://search.google.com/search-console
2. Нажмите "Добавить свойство"
3. Выберите тип: "URL-адрес"
4. Введите: https://flgmn.tech
5. Нажмите "Продолжить"

### Шаг 2: Верификация сайта
#### Способ 1: HTML тег (рекомендуется)
1. Выберите способ верификации "Теги HTML"
2. Скопируйте meta-тег
3. Добавьте в `app/layout.tsx` в раздел `head`:
```tsx
<meta name="google-site-verification" content="YOUR_VERIFICATION_CODE_HERE" />
```
4. Нажмите "Проверить"

#### Способ 2: HTML файл
1. Скачайте файл верификации
2. Загрузите его в `/public/`
3. Нажмите "Проверить"

### Шаг 3: Отправка Sitemap
1. Перейдите в "Карты сайта"
2. Нажмите "Добавить карту сайта"
3. Введите: https://flgmn.tech/sitemap.xml
4. Нажмите "Отправить"

### Шаг 4: Проверка индексации
1. Перейдите в "Охват"
2. Проверьте количество индексированных страниц
3. Исправьте ошибки, если они есть

### Шаг 5: Мониторинг
Регулярно проверяйте:
- **Производительность** - CTR и позиции в выдаче
- **Охват** - ошибки индексации
- **Улучшения** - Mobile Usability, Rich Results
- **Ссылки** - внешние ссылки на сайт

## Яндекс.Вебмастер

### Шаг 1: Добавление сайта
1. Перейдите на https://webmaster.yandex.ru
2. Нажмите "Добавить сайт"
3. Введите: https://flgmn.tech
4. Выберите сертификат владения
5. Нажмите "Добавить"

### Шаг 2: Верификация
#### Способ 1: HTML тег (рекомендуется)
1. Выберите "Тег Яндекса"
2. Скопируйте код
3. Добавьте в `app/layout.tsx`:
```tsx
<meta name="yandex-verification" content="YOUR_YANDEX_CODE_HERE" />
```
4. Подтвердите владение

#### Способ 2: CNAME запись
1. Выберите "CNAME"
2. Добавьте DNS запись в ваш провайдер
3. Подождите 24-48 часов
4. Подтвердите владение

### Шаг 3: Загрузка Sitemap
1. Перейдите в "Карты сайта"
2. Нажмите "Добавить карту сайта"
3. Введите: https://flgmn.tech/sitemap.xml
4. Нажмите "Отправить"

### Шаг 4: Анализ трафика
1. Перейдите в "Поиск → Статистика"
2. Посмотрите популярные запросы
3. Оптимизируйте контент для ТОП запросов

### Шаг 5: Мониторинг
Следите за:
- **Исключение URL** - почему страницы не индексируются
- **Статистика сканирования** - как часто робот посещает сайт
- **Поиск** - позиции и клики по запросам

## Рекомендации по содержанию

### Для лучшего ранжирования:
1. **Уникальный контент** - минимум 1000-2000 слов на статью
2. **Внутренние ссылки** - связывайте релевантные страницы
3. **Внешние ссылки** - получайте ссылки из авторитетных источников
4. **Обновляйте контент** - регулярно обновляйте старые статьи
5. **Структурированные данные** - используйте schema.org разметку
6. **Быстрая загрузка** - оптимизируйте скорость страниц
7. **Мобильная оптимизация** - проверьте мобильную версию

## Типичные ошибки, которых нужно избежать

❌ **НЕ делайте:**
- Не используйте черные шляпы SEO (клоакинг, скрытые ссылки)
- Не дублируйте контент на разных страницах
- Не создавайте тысячи страниц с тонким контентом
- Не используйте автоматически сгенерированный контент
- Не покупайте ссылки или не участвуйте в линк-схемах
- Не скрывайте ссылки CSS или другими способами
- Не создавайте дверные страницы (doorway pages)
