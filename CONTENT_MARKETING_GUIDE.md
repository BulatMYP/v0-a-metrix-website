## 📊 Метрики для отслеживания

Для каждой статьи отслеживайте:

- **Views** - количество просмотров
- **Avg. Time on Page** - среднее время на странице (цель: >2 минут)
- **Bounce Rate** - процент отскоков (цель: <50%)
- **Conversions** - сколько людей кликнули на CTA
- **SEO** - позиция в поиске для основного ключевого слова
- **Social shares** - количество шеров в соцсетях

## ✍️ SEO советы для контента

1. **Meta Description** (155-160 символов):
   - Должна содержать основное ключевое слово
   - Должна содержать CTA ("Узнайте", "Читайте", "Скачайте")

2. **H1 (Page Title)**:
   - Содержит главное ключевое слово
   - 50-60 символов

3. **URL slug**:
   - Короткий и описательный
   - Например: `/blog/zapusk-tech-startupa`

4. **Изображения**:
   - Alt text с ключевыми словами
   - Сжатые для быстрой загрузки
   - Минимум 1 изображение на статью
